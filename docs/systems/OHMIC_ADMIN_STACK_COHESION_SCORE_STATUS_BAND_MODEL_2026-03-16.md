# Ohmic Admin Stack Cohesion Score Status Band Model

Date: 2026-03-16
Project: ohmic

## Bands

- `healthy`: stack stays inside one clear filing cluster
- `watch`: one adjacent branch is present but still explainable
- `pressure`: branch drift or fallback noise is increasing
- `critical`: stack no longer has a clear continuity story

This score is for operator interpretation, not automatic claim mutation.
