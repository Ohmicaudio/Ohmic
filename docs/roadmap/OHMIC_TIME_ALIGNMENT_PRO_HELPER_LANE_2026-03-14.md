# Ohmic Time Alignment Pro Helper Lane

## Decision

`Time Alignment` should not be part of the next public mini-tool wave.

It should live as a Pro/helper lane tied to:

- tuning workflows
- handheld helper surfaces
- hardware-linked setup flows

## Why

Unlike the strongest public mini-tools, `Time Alignment` is not mostly a simple search-entry utility.

Its value increases when it sits near:

- measured speaker distances
- tuning sequences
- DSP setup steps
- hardware-aware commissioning flows

That makes it more valuable as a helper inside real workflow than as a broad public lead-gen calculator.

## Best Future Surfaces

Recommended first consumers:

1. tuning workflow helpers in `ohmic-audio-labs`
2. handheld/remote helper views
3. future hardware-linked setup and commissioning flows

Existing signal:

- there is already a `TimeAlignmentHelper.tsx` surface in the main app

That supports the idea that this lane is naturally a workflow helper, not just a general public calculator.

## Why It Stays Out Of Second Public Wave

The second public wave is better used for:

- one generic electrical utility
- one design/SPL utility
- one upgrade-comparison utility

`Time Alignment` is more specialized and gets better when paired with:

- speaker layout context
- tuning workflow context
- device and setup context

## Product Role

Best role:

- Pro helper
- tuning-side assistant
- handheld support utility

Not best role:

- broad public entry tool
- SEO-first mini-tool

## Rule

Do not force `Time Alignment` into the public mini-tool ladder just because it already exists.

Keep it where it can become more workflow-aware and more useful.
