## Extracted Prompt Briefs

### Bandpass Enclosure Blueprint

Create a dark-theme engineering blueprint of a 4th-order bandpass subwoofer enclosure for car audio. Show side, top, and front cutaway views with chamber labels, driver position, port path, panel thickness, internal dimensions, net volumes for sealed and vented chambers, tuning callout, airflow arrows, and a compact notes block. Style: clean CAD-like lines, white/ice-blue labels on dark charcoal, no photorealism, no unnecessary gradients.

### Dual Battery Parallel Wiring

Create a clear electrical diagram for a 12 V dual-battery parallel car audio system. Include under-hood starter battery, rear auxiliary battery, alternator, chassis grounds, fuse protection near each positive terminal, distribution blocks, amplifier load, optional isolator/BMS note, and cable gauge callouts. Style: installer-grade diagram, high contrast, dark theme, colorblind-safe line coding, labeled current flow arrows.

### Big Three Upgrade Diagram

Create an installer reference diagram showing the Big Three upgrade in a vehicle: alternator positive to battery positive with fuse, battery negative to chassis, and engine block to chassis. Show OEM paths vs upgraded paths, recommended cable gauge, fuse locations, and clean before/after topology. Style: technical infographic, minimal, precise, dark background.

### Full Electrical System Overview

Create a system-level topology diagram of a high-power car audio electrical system. Include alternator, front battery, rear battery bank, fuse blocks, distribution, amplifiers, DSP/head unit branch, grounds, and optional capacitor/ultracap module. Make it a flagship reference diagram with hierarchy, current paths, and service checkpoints.

### Crossover Frequency Guide

Create a quick-reference chart for typical car audio crossover starting points. Include subwoofer, midbass, midrange, tweeter, and common slope/alignment suggestions. Show frequency bands on a horizontal spectrum with recommended overlaps and notes for daily driver vs loud/SPL-biased setups.

### DVC Wiring Configurations

Create a clean diagram sheet showing all practical dual-voice-coil subwoofer wiring options for common coil impedances. Include series, parallel, separate-channel cases, resulting impedance, and do/do-not notes. Style: white line diagrams with clear terminal polarity, dark theme, compact but legible.

### Enclosure Quick Design Card

Create a one-page quick-reference card for enclosure design. Include net vs gross volume, displacement checklist, panel thickness reminder, bracing reminder, sealing checklist, and port area/tuning reminders. Style: printable field card with strong typographic hierarchy.

### Filter Slope Comparison

Create a comparison chart of 6, 12, 18, 24, and 48 dB/octave slopes using normalized response curves on one graph. Add brief annotations about overlap, protection, phase behavior, and typical use cases in mobile audio.

### Fuse Type Reference

Create a visual chart comparing ANL, mini-ANL, MIDI, ATO/ATC, and maxi fuse types for car audio/electrical systems. Show each fuse silhouette, typical current ranges, best-use cases, and installation notes.

### Ground Point Preparation

Create a step-by-step visual card showing how to prepare a proper chassis ground point for car audio. Include paint removal, bare metal prep, ring terminal, fastener type, anti-corrosion finishing, and bad example vs good example.

### Group Delay Example

Create an annotated group-delay graph for car audio explaining what low, moderate, and excessive group delay look like in a subwoofer system. Mark frequency axis, milliseconds axis, and example interpretations near tuning and crossover regions.

### Impedance Curve Guide

Create a guide graphic explaining loudspeaker impedance curves. Show resonance peak, nominal impedance region, box/port effects, and why impedance is not flat. Include sealed vs ported comparison inset.

### Phase Response Interpretation

Create an educational graph sheet that explains phase response in audio. Show phase traces for typical crossover/filter examples, wrap/unwrap explanation, and notes on why phase alone is not enough without magnitude/context.

### Ported Box Blueprint

Create a dimensioned blueprint for a practical 12-inch ported subwoofer enclosure with front, top, and side views; slot-port geometry; panel thickness; gross/net volume; tuning frequency; airflow direction; and material notes. Dark theme, engineering style.

### Sealed Box Blueprint

Create a dimensioned blueprint for a practical 12-inch sealed subwoofer enclosure with front, top, and side views; panel thickness; internal clear dimensions; target net volume; driver displacement note; and brace locations. Dark theme, engineering style.

### Single Amp Basic Wiring

Create a beginner wiring diagram for a single-amplifier car audio system with battery, fuse, amplifier, ground, RCA/signal input, speaker outputs, and remote turn-on. Show correct routing separation between power and RCA where possible.

### Speaker Anatomy Cross-Section

Create a labeled cross-section of a dynamic loudspeaker showing dust cap, cone, surround, spider, voice coil, former, pole piece, top plate, magnet, basket, terminals, and venting. Style: technical cutaway, not cartoonish.

### System Design Quick Reference

Create a one-page system design card for car audio planning. Include goals, power budget, electrical needs, enclosure needs, tuning path, measurement path, and common failure checkpoints.

### Audio Spectrum Reference

Create a horizontal audio spectrum chart from 20 Hz to 20 kHz with labeled regions for sub-bass, bass, low midrange, midrange, upper midrange, presence, brilliance, and common instrument/voice anchors. Dark theme, crisp labels.

### TS Parameter Measurement Diagram

Create a measurement workflow diagram for Thiele-Small parameter testing. Include driver, jig, resistor, measurement software, free-air test, added-mass or test-box step, and resulting parameters. Clean lab-style layout.

### Three-Amp Active System

Create a wiring topology diagram for a three-amplifier fully active front stage plus subwoofer system. Include source, DSP, amplifier channels, speaker outputs, subwoofer channel, power distribution, and grounds. Clear channel labeling is critical.

### Two-Amp Front Plus Sub System

Create a wiring diagram for a two-amplifier car audio system with one full-range/front-stage amp and one sub amp. Include battery, fusing, distribution, grounds, source/DSP path, and speaker/sub connections.

### Waterfall / CSD Reading Guide

Create an annotated waterfall/CSD plot guide that explains decay ridges, ringing, resonances, and what a cleaner decay looks like. Include readable callouts for beginners.

### Time Alignment Diagram

Create a car cabin top-view diagram showing left/right speaker positions relative to driver ears, path-length differences, and conversion from distance difference to milliseconds for DSP time alignment.

### Microphone Placement Standard

Create a measurement-procedure card showing microphone placement examples for near-field, far-field, listening position, A-pillar SPL measurement, and subwoofer ground-plane measurement. Include "when to use each" notes.
