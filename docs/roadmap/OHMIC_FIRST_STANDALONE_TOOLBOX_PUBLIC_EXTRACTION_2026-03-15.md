# Ohmic First Standalone Toolbox Public Extraction

## Decision

The first standalone public mini-tool extraction candidate should be:

- `Ohm's Law`

## Why It Wins

### 1. Broadest immediate search intent

`Ohm's Law` has the clearest general-purpose utility and the easiest public
search-entry story.

It is useful to:

- hobbyists
- installers
- beginners
- anyone doing basic electrical sanity checks

### 2. Strongest fit for the free offer ladder

It works cleanly as:

- a public utility
- an educational entry point
- a low-friction top-of-funnel page

That aligns well with the `Ohmic Free` positioning in
`OHMIC_OFFER_LADDER_AND_POSITIONING_2026-03-14.md`.

### 3. Cleanest boundary

It has a strong boundary from deeper lanes:

- it is not `Wiring Lab`
- it is not BassBuilder
- it is not Pro-only measurement or tuning logic

That makes it safer to publish standalone without confusing the product map.

### 4. Already implemented in the toolbox surface

The shared math exists and the in-toolbox panel now exists.

That means the extraction candidate is no longer theoretical.

## Why The Other Candidates Wait

### Cone Area

Keep inside the toolbox longer.

Reason:

- useful, but more naturally adjacent to design and BassBuilder flows
- better as a supporting comparison utility than the first broad standalone
  public page

### Acoustic Gain

Keep inside the toolbox longer.

Reason:

- strong utility, but its explanation quality matters more
- easy to oversimplify publicly without enough educational framing

## Extraction Preconditions

Do not extract `Ohm's Law` into a standalone public page until these are true:

1. the in-toolbox version is stable
2. shared math fixtures are trusted
3. the page copy clearly distinguishes it from `Wiring Lab`
4. the public page has a clear CTA path back into the toolbox or broader Ohmic
   tool ecosystem
5. the page gets a proper SEO title, description, and internal-link role

## Recommended Public Positioning

Use the standalone page as:

- a simple electrical calculator
- a teaching surface
- an entry page into broader Ohmic utility tools

Do not position it as:

- a full speaker matching tool
- an amp safety oracle
- a replacement for the smarter `Wiring Lab` lane

## Recommended Next Step

After the current toolbox wave settles:

- define the standalone page copy block
- define its CTA path
- define its route and canonical placement

That should happen before building any second standalone extraction.

## Summary

`Ohm's Law` should be first because it is:

- the easiest to understand
- the broadest in search value
- the cleanest in product boundary
- the best free-entry utility candidate
