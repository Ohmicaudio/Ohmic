# Define System Created Claim Origin Report

## Goal

Define the worker-facing report that explains when an active claim was created
by the system or automation path rather than by the current worker.

## Focus

- claim origin
- owner identity
- linked task
- reason
- timestamping

## Acceptance

- one system-claim-origin packet is explicit
- workers can distinguish self-created and system-created claims clearly
