# Define Administrator Action State Restriction Matrix

## Goal

Define the restriction matrix that limits which administrator actions are valid
from which intake lifecycle states.

## Focus

- current-state to action validity
- reopen requirements
- archive restrictions
- waiting-state restrictions
- overlay-added disallow rules

## Acceptance

- one state-restriction packet is explicit
- invalid transitions are separated from generic action availability
