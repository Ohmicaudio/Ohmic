## Goal

Define the audit event family for applied, removed, accepted, or ignored tag
assignments.

## Focus

- tag assignment events
- source attribution
- accepted default versus accepted hint
- removal events
- timestamps

## Acceptance

- one tag-audit packet is explicit
- tag history remains reconstructable and attributable
