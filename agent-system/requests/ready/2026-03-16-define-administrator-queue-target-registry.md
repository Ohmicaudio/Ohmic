# Define Administrator Queue Target Registry

## Goal

Define the stable registry for queue targets that administrator actions can
route into.

## Focus

- canonical queue target ids
- display labels
- overlay visibility
- allowed-action bindings
- deprecated target handling

## Acceptance

- one queue-target registry packet is explicit
- route actions can validate against stable destination ids
