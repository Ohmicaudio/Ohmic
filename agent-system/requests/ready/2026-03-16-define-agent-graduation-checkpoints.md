# Define Agent Graduation Checkpoints

## Goal

Define the checkpoints that let an agent graduate from fresh to trusted based
on repeated live-task performance rather than vibes.

## Focus

- completion count
- correctness rate
- reopen/rework rate
- verification discipline
- local-context retention

## Acceptance

- graduation checkpoints are explicit
- trust growth is tied to outcomes
- agent onboarding becomes measurable
