# Define Live Task Route Training Lane

## Goal

Define how new agents should be trained through bounded real tasks in actual
project context instead of relying only on handoffs and static briefings.

## Focus

- bounded real tasks
- verification requirements
- correction loop
- repeated exposure to local naming and boundary rules

## Acceptance

- one route-training model is defined
- handoff-only onboarding is no longer treated as sufficient
- the system has a repeatable way to teach the route
