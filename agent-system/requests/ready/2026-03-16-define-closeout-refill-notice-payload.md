# Define Closeout Refill Notice Payload

## Goal

Define the payload shown to a worker when a family is refilled during closeout
of a packet in that same family.

## Focus

- closing task id
- refill event id
- new child count
- actor attribution
- timestamp

## Acceptance

- one closeout-refill payload packet is explicit
- refill-during-closeout gets a stable worker-facing shape
