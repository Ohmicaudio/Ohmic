# Define Administrator Inactive Intake Visibility Projection

## Goal

Define the projection used to expose archived and routed intake items when a
desk or audit view intentionally includes them.

## Focus

- archived visibility
- routed visibility
- reopen affordance fields
- filtered versus default views
- summary labels

## Acceptance

- one inactive-visibility packet is explicit
- inactive items can appear intentionally without polluting the default queue
