# Define Administrator Normalization Warning Review Lane

## Goal

Define the desk behavior for intake items that normalized with warnings or low
confidence instead of cleanly.

## Focus

- warning badges
- manual review queue
- confidence display
- operator correction path
- escalation triggers

## Acceptance

- one warning-review packet is explicit
- low-confidence intake does not disappear into normal queue flow
- the desk has a bounded manual review lane
