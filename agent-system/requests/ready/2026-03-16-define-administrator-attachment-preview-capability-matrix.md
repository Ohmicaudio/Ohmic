# Define Administrator Attachment Preview Capability Matrix

## Goal

Define which attachment kinds can produce which preview kinds in the first
administrator shell.

## Focus

- preview kind by MIME family
- preview generation rules
- unsupported asset behavior
- cold retrieval fallback
- warning states

## Acceptance

- one preview-capability packet is explicit
- preview expectations are decoupled from raw MIME guessing
