# Open Questions

Use this file for unresolved questions that should stay visible across agents and sessions.

## Purpose

- keep important unanswered questions out of chat-only memory
- give agents one canonical place to check for questions that may unblock or redirect work
- make it easy to notice new questions after a meaningful completed task

## What Belongs Here

- cross-project routing questions
- unresolved blockers without a clean owner yet
- questions another agent should see before continuing adjacent work
- decisions that may change the next safe step

## What Does Not Belong Here

- ordinary finished requests
- active edit claims
- long decision histories
- one-off chat curiosities with no operational consequence

## Usage Rule

- append new open questions as short dated bullets
- remove or mark resolved items when the answer is known
- if a question turns into a concrete task, create a request record and link it
