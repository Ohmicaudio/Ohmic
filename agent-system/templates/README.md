# Templates

These templates exist to keep shared agent docs consistent in shape and language.

Use them as starting points, not rigid forms.

Preferred order:

1. pick the template closest to the document type
2. remove sections that do not apply
3. keep section names stable where possible

Available templates:

- `short-term-template.md`
- `project-overlay-template.md`
- `handoff-template.md`
- `request-template.md`
- `job-claim-template.md`
