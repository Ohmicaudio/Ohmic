# Handoffs

This folder stores concise session handoffs.

## Rules

- use one file per session topic when needed
- prefer `YYYY-MM-DD-topic.md`
- keep handoffs factual and short
- promote durable facts upward after the session
- use stable sections so handoffs are easy to scan and index later

## Reminder

Handoffs are for continuity, not authority.

Start from `templates/handoff-template.md` when creating a new handoff.
